# 🫀 Guía RCP - Asistente de Emergencia y Primeros Auxilios

**Guía RCP** es una aplicación móvil desarrollada en React Native y Expo, concebida para ser un asistente universal en situaciones de paro cardíaco y emergencia médica. 

Diseñada sin barreras ni elementos bloqueados, permite a cualquier usuario (adultos o niños) recibir instrucciones por voz y un metrónomo de ritmo constante durante las compresiones.

## 🚀 Características
* 🔊 **Guía por Voz Nativa (Offline):** Funciona mediante síntesis de voz sin requerir conexión a internet.
* 🫀 **Metrónomo Háptico (110 BPM):** Emite un ritmo visual y pulsaciones en el teléfono para mantener la frecuencia correcta de RCP.
* ⚡ **Acceso Directo:** Sin menús innecesarios ni registros de usuario.

## 🛠️ Desarrollo Local
```bash
# Clona este repositorio
git clone [https://github.com/joseluis1212/Guia-RCP.git](https://github.com/joseluis1212/Guia-RCP.git)
cd Guia-RCP

# Instala las dependencias
npm install

# Inicia la aplicación en Expo
npx expo start
